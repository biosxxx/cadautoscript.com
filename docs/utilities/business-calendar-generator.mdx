# Business Calendar Generator

Build a clean yearly business calendar with regional holidays, company branding, and export-ready layouts.

<a className="button button--primary" href="/utilities/business-calendar-generator/" data-nobrokenlinkcheck>Open utility</a>

## What it does
- Builds a full-year calendar from a selected country and year.
- Tracks vacation, sick days, and custom workdays in a personalization panel.
- Supports logo uploads and theme/backdrop styling.
- Exports print-ready PDF or high-resolution PNG files.

## When to use it (and when not to)
**Use it when:**
- You need a holiday-aware yearly calendar for staffing or project planning.
- You want a clean PDF/PNG export for sharing with teams or clients.
- You need a single page calendar with branding and working-hour context.

**Avoid it when:**
- You need live integration with HR systems or payroll.
- You need multi-year or multi-site scheduling workflows.

## Inputs and outputs
### Inputs
| Input | Description |
| --- | --- |
| Country | Sets the holiday dataset for the calendar. |
| Year | Generates the calendar for the selected year. |
| Hours per day | Used in personalization summaries and day annotations. |
| Personalization | Mark vacation, sick leave, or custom work days. |
| Branding | Upload a company logo and pick theme/backdrop styles. |
| Export settings | Choose paper size and PDF quality. |

### Outputs
| Output | Format | Notes |
| --- | --- | --- |
| Business calendar | On-screen | Full-year view with holidays and annotations. |
| PDF export | .pdf | Print-ready, selectable quality. |
| PNG export | .png | High-resolution raster export. |

## How to use
1. Select a **country** and **year** in the left panel.
2. Set **hours per day** to match your working schedule.
3. Open **Personalization** to add vacation or sick days.
4. Upload a logo and adjust the theme/backdrop if needed.
5. Export as **PDF** or **PNG** using the export buttons.

**Example:** Generate a 2026 calendar for Germany, add your company logo, mark vacation weeks, then export an A4 PDF for planning meetings.

## Accuracy and verification
- Holiday data is preloaded; verify against local regulations or company policies.
- Exports are generated locally in the browser.
- Personalization data is stored in local storage and can be cleared any time.

## FAQ
- **Does it run offline?** Yes. Once loaded, generation and exports run locally.
- **Can I import or export personalization data?** The personalization panel supports markdown import/export.
- **Will it upload my logo or data?** No. Files and data stay in your browser.
- **Which paper sizes are supported?** A-series sizes (A4/A3/A2) with adjustable quality.
